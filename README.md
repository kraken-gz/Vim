# Vim
My Vim setup

**Windows**:

Global settings 

```C:\Program Files (x86)\Vim``` 

Local user settings 

```C:\Users\<user>\vimfiles``` 

--------------------------------------------------------------------------------

**Linux**:

Global settings 

```/etc/vim ```

Local user settings 


```~/.vimrc ```


Put theme in ```/usr/share/vim/vim*/colors```

```sudo cp atom-dark-256.vim /usr/share/vim/vim*/colors```
